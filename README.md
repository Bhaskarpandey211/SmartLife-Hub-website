# SmartLife-Hub-website

SmartLife Hub – Modern Lifestyle & Services Website

Create a fully responsive SmartLife Hub website using only HTML and CSS. The website should provide information about lifestyle services such as fitness, education, travel, shopping, and technology. The project should have a clean, modern, and user-friendly interface while demonstrating all the essential concepts of HTML and CSS.

Project Objective:-

The objective of this project is to build a professional multi-section website that helps beginners practice and understand the most important HTML and CSS concepts through a single real-world project. The website should be responsive, visually appealing, and well-structured using semantic HTML elements and modern CSS techniques.

Features to Include:-

1. Responsive Header with Logo and Navigation Menu
2. Hero Section with Background Image and Call-to-Action Button
3. About Section
4. Services Section using CSS Grid
5. Featured Products or Courses Cards
6. Statistics Section
7. Pricing Plans
8. Team Members Section
9. Customer Testimonials
10. Image Gallery
11. Video Showcase
12. Frequently Asked Questions (FAQ)
13. Contact Form
14. Google Map
15. Newsletter Subscription
16. Footer with Social Media Links
